## Featured Work

### Manip Tool Hub

[Manip Tool Hub](https://racingemperor.github.io/manip-tool-hub/) is my public
TypeScript and Next.js project for browsing embodied-intelligence tools across
perception, cognition, reasoning/planning, and execution/control.

What it shows:

- A searchable tool catalog with capability, readiness, and resource filters
- Tool detail pages with task summaries, input/output contracts, examples,
  parameters, benchmark rows, demos, and official links
- A typed data registry so new tool entries can be added without hand-wiring pages
- A static GitHub Pages deployment with screenshots and reproducible setup notes

Current boundary: this project is a tool-hub and documentation/validation
interface. I am extending it toward runnable local validation examples with
input/output artifacts, logs, and failure cases.

### RoleFit Pipeline

[RoleFit Pipeline](https://github.com/racingemperor/rolefit-pipeline) is a
Python workflow experiment for early-career role fit, learning gaps, and resume
reverse design. It explores how to turn job goals into structured analysis,
project recommendations, and resume-ready evidence checks.
